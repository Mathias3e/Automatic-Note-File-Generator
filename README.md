- `C:\Users\<user name>\AppData\Roaming` | benutzereinstellungen
- `C:\Users\<user name>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs` | Programverlinkung
- `C:\Users\<user name>\AppData\Local\Programs\` | Programcode
